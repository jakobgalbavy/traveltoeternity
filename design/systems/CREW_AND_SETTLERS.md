# Crew & Settlers

## Distance × Specialization Curve

Longer missions require more specialized crew — and specialists cost more. This creates a natural cost escalation that eats into the distance payout bonus, keeping risk/reward balanced.

| Distance tier | Crew requirement | Example premium role |
|---------------|-----------------|---------------------|
| Short-range | Generalists fine | Basic engineer |
| Mid-range | 1–2 specialists needed | Certified botanist, trauma medic |
| Long-range | Full specialist manifest | Radiation biologist, deep-space navigator |
| Extreme | Rare/unique specialists | Cryogenics engineer, exobiologist |

Specialists are harder to hire (limited pool), more expensive, and may have demands (better quarters, hazard pay). Failing to crew appropriately raises event failure chances dramatically.

## Two Populations

### Crew
Specialists hired before departure. Actively managed, have skills, can be assigned to tasks.

| Role | Primary skill | Secondary |
|------|--------------|-----------|
| Commander | Leadership (morale events) | Navigation |
| Engineer | Hull / mechanical repair | Power systems |
| Botanist | Crop health / grow bays | Medicine |
| Medic | Illness / injury treatment | Morale counseling |
| Navigator | Course correction / hazards | Comms |
| Security | Conflict resolution | Intimidation / de-escalation |

Crew have stats (skill level, health, stress) and personality traits that affect event outcomes and inter-crew relationships.

### Settlers
Passengers — the reason for the mission. Not directly commanded, but their collective morale affects the ship.

- Represented individually (name, background, needs, quirks) — OPEN DECISION O-005
- Form informal factions if left unmanaged
- Can be pressed into unskilled labor in emergencies (low efficiency)
- Deaths reduce final payout and reputation

## Morale System

Morale is tracked separately for Crew and Settlers (both matter).

### Morale Drivers
| Factor | Crew | Settlers |
|--------|------|----------|
| Food quality & quantity | ✓ | ✓ |
| Living space | ✓ | ✓ |
| Safety (recent events) | ✓ | ✓ |
| Clear leadership | ✓ | — |
| Entertainment / downtime | ✓ | ✓ |
| Knowing destination is closer | — | ✓ |
| Crew death | ✓ | ✓ |
| Unresolved conflicts | ✓ | ✓ |

### Morale Thresholds
- **High (75–100%):** Bonus efficiency. Settlers self-organize, help out.
- **Stable (40–74%):** Normal operations.
- **Low (20–39%):** Work slowdowns, event frequency rises, conflicts more likely.
- **Critical (<20%):** Mutiny risk. Settlers may take over ship. Mission failure possible.

## Individual Settler Traits (if O-005 = individuals)

Each settler has:
- **Background** (farmer, technician, teacher, soldier, criminal, idealist...)
- **Need** (safety / belonging / autonomy / purpose)
- **Quirk** (hoarder / peacemaker / agitator / hypochondriac / loner...)
- **Relationship map** (who they like/dislike)

Trait combinations create emergent drama: a soldier with autonomy needs will chafe under strict rationing; a hoarder during a food crisis becomes a flashpoint.
